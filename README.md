# Gestión de Datos de Países en Python

**Trabajo Práctico Integrador - Programación 1 - UTN**

## Descripción
Sistema de gestión de información sobre países desarrollado en Python.
Permite realizar búsquedas, filtros, ordenamientos y estadísticas sobre
un dataset de países cargado desde un archivo CSV.

## Integrantes
- Avila Hernan - funciones.py / paises.csv
- Amoroso Juan - main.py / README.md

## Estructura del proyecto

gestion-paises-python/

├── main.py         → Menú principal del programa
├── funciones.py    → Funciones de búsqueda, filtros y estadísticas
├── paises.csv      → Dataset con 36 países
└── README.md       → Documentación del proyecto

## Requisitos
- Python 3.x

## Cómo ejecutar el programa
1. Clonar el repositorio
2. Abrir una terminal en la carpeta del proyecto
3. Ejecutar: `python main.py`

## Opciones del menú
1. Agregar un país
2. Actualizar un país
3. Buscar país por nombre
4. Filtrar por continente
5. Filtrar por rango de población
6. Filtrar por rango de superficie
7. Ordenar países
8. Mostrar estadísticas
0. Salir
